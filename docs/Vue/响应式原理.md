# 响应式原理

对比Vue2和Vue3的响应式原理，及各自的优缺点

## Vue2

VitePress provides Syntax Highlighting powered by [Shiki](https://github.com/shikijs/shiki), with additional features like line-highlighting:

**Input**

````md
```js{4}
export default {
  data () {
    return {
      msg: 'Highlighted!'
    }
  }
}
```
````

**Output**

```js{4}
export default {
  data () {
    return {
      msg: 'Highlighted!'
    }
  }
}
```

## Vue3

**Input**

```md
::: info

:::

:::

::: warning

::: danger

:::

::: details

:::
```

::: info
This is an info box.
:::

::: tip
This is a tip.
:::

::: warning

::: danger
